Data: Janeiro de 2026

Responsável: Lívia Silva  

## Introdução
Este relatório apresenta o processo de implementação de serviços em nuvem utilizando a plataforma AWS, desenvolvido como parte de um desafio prático da DIO (Digital Innovation One).

O projeto teve como objetivo compreender, na prática, como os serviços da AWS podem ser utilizados para melhorar a organização, a disponibilidade e a eficiência de recursos tecnológicos, além de reduzir custos com infraestrutura física.

---

## Descrição do Projeto
O projeto de implementação de serviços AWS foi dividido em 3 etapas, cada uma com seus objetivos específicos, conforme descrito abaixo.

### Etapa 1
- **Nome da ferramenta:** Amazon EC2  
- **Foco da ferramenta:** Computação em nuvem  
- **Descrição de caso de uso:**  
  Utilização do Amazon EC2 para simular a criação de servidores virtuais, permitindo hospedar aplicações e serviços de forma escalável e sob demanda.

### Etapa 2
- **Nome da ferramenta:** Amazon S3  
- **Foco da ferramenta:** Armazenamento de dados  
- **Descrição de caso de uso:**  
  Uso do Amazon S3 para armazenamento seguro de arquivos e dados do projeto, garantindo alta disponibilidade e durabilidade das informações.

### Etapa 3
- **Nome da ferramenta:** AWS IAM  
- **Foco da ferramenta:** Gerenciamento de acesso e segurança  
- **Descrição de caso de uso:**  
  Implementação do AWS IAM para controle de permissões e usuários, reforçando a segurança e boas práticas no acesso aos recursos em nuvem.

---

## Conclusão
A implementação dos serviços AWS neste projeto acadêmico demonstrou como a computação em nuvem pode trazer benefícios como escalabilidade, segurança e redução de custos operacionais.

A utilização dessas ferramentas contribui para maior eficiência e organização dos processos, sendo recomendada a continuidade do aprendizado e aprofundamento nos serviços AWS.

## Descrição

Este projeto foi desenvolvido como parte de um desafio prático da plataforma DIO (Digital Innovation One), com o objetivo de aplicar os conceitos estudados sobre computação em nuvem utilizando serviços da AWS.

Durante o desenvolvimento do projeto, foram abordados serviços como Amazon EC2, Amazon S3 e AWS IAM, simulando um cenário acadêmico de implementação em nuvem. O foco foi compreender como essas ferramentas podem contribuir para escalabilidade, segurança e melhor organização de recursos tecnológicos.

O projeto tem caráter educacional e visa fortalecer o portfólio técnico, além de consolidar o aprendizado prático em cloud computing.
